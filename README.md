# COS for Web Project

> A Vue.js project

##Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```


For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).

## 技术栈

Vue + vue-router + Vuex + Axios + Webpack

## 新安装npm模块

#### bootstrap
bootstrap模块
安装：npm install bootstrap@3.3.0 --save-dev

####http-proxy-middleware
http反向代理中间件
安装：npm install http-proxy-middleware@0.17.4 --save-dev

#### scss scss-loader
安装：npm install scss sass --save-dev
安装：npm install sass-loader node-sass --save-dev
安装：npm install scss-loader node-scss --save-dev
